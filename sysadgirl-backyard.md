# シチュエーション案1

「改竄の検出」にフォーカスしてみる

 * 出自の分からない怪しい
   インストールイメージが
   すでに手元にある
 * 公式サイトで公開されてる
   ハッシュ値と比較すれば安心！

→そんな場面なさそう

# シチュエーション案2

 * 受領したでかいファイルが
   本物かどうか確かめたい

→そもそも電子署名か暗号化が
